# DO SPORTS Homepage v3

GitHub + Netlify 배포용 DO SPORTS 공식 홈페이지입니다.

## 포함 페이지

- `/` : DO SPORTS 공식 홈페이지
- `/link` : 상담 및 신청서 페이지 (`https://dosportslink.netlify.app/` iframe)
- `/schedule` : 실시간 시간표 페이지 (`https://dosportstt.netlify.app/` iframe)
- `/admin` : 시간표 관리자 페이지로 이동
