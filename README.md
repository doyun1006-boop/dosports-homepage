# DO SPORTS 홈페이지 v1

GitHub + Netlify 연결용 DO SPORTS 공식 홈페이지 파일입니다.

## 포함 페이지

- `/` : DO SPORTS 공식 홈페이지
- `/link` : 상담 및 신청서 페이지 (`https://dosportslink.netlify.app/` iframe 연결)
- `/schedule` : 실시간 시간표 페이지 (`https://dosportstt.netlify.app/` iframe 연결)
- `/admin` : 시간표 관리자 페이지로 이동 (`https://dosportstt.netlify.app/admin`)

## GitHub 업로드 구조

저장소 루트에 아래 파일/폴더를 올려주세요.

```text
public
netlify.toml
README.md
```

Netlify 설정:

- Build command: 비워두기
- Publish directory: `public`
